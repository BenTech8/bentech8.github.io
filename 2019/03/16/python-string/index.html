String

发表于 2019-03-16 更新于 2025-03-16 分类于 Python 阅读次数： Valine：
本文字数： 4.7k 阅读时长 ≈ 8 分钟

简介

  • 把字符连成串，在python中用’, ‘’, ‘’’, “””引起来的内容被称为字符串
  • 字符串是不可变对象，所有操作都是产生新字符串返回

切片和索引

索引

索引就是下标，下标从0开始

切片

  • 使用下标来截取部分字符串的内容
  • 语法：str[start: end]
  • 规则：顾头不顾尾，从start 开始截取，截取到end位置，但不包括end

步长：

步长如果是正数，则从左往右取，如果是负数，则从右往左取。默认为1

方法

大小写

  • capitalize

功能：对第一个单词首字母大写，对于非第一个单词首字母进行检查，若大写则改为小写。

  • lower

功能：将字符串全部转换成小写

  • upper

功能：将字符串全部转换成大写

  • swapcase

功能：大小写互相转换

  • casefold

功能：转换成小写

和lower的区别：lower()对某些字符支持不够好。casefold()对所有的字母都有效。比如东欧的一些字母。

  • title

功能：每个被特殊字符隔开的字母首字母大写

切来切去

  • center

功能：居中

  • expandtabs

功能：更改tab的长度，默认长度改为8

  • strip

功能：默认去掉左右两端的空白(空格，\n, \t)

  • lstrip

功能：默认去掉左端的空白(空格，\n, \t)

  • rstrip

功能：默认去掉右端的空白(空格，\n, \t)

  • replace

功能：字符串替换

  • split

功能：字符串切割

  • 当括号内为无时即s.split()，此时根据空白切割
  • s.split(“|”, 1),根据字符串s中第一个”|”切割
  • join

功能：将列表名元素按给定字符连接

查找

  • startswith

功能：判断是否以某个/些字符开头

  • endswith

功能：判数是否以某个/些字符结尾

  • count

功能：查找某个/些字符出现的次数

  • find

功能：查找某个/些字符出现的位置，如果没有返回-1

如果查找的某个字符在字符串中出现多次，则查找所得的结果是第一个字符出现的位置。

切片找：

  • rfind

从字符串末尾开始查找

  • index

功能：求索引位置，如果找不到索引，程序会报错

  • rindex

从字符串末尾开始查找

格式化输出

  • format

条件判断

  • isalnum

功能：是否由字母和数字(int)组成，可以理解为包含isalpha和isnumeric的功能

  • isalpha

功能：是否由字母组成

  • isdigit

功能：是否由阿拉伯数字(%d)组成

  • isdecimal

  • isnumeric

计算字符串长度

  • len

len()是python的内置函数。

迭代

  • 可以使用for循环来遍历字符串中的每一个字符
  • 可迭代对象：可以一个一个往外取值的对象
