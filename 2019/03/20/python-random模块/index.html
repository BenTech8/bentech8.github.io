random

所有关于随机相关的内容都在random模块中

相关功能

random()

  • 产生(0, 1)之间的小数

uniform(a, b)

  • 产生(a, b)之间的小数

randint(a, b)

  • 产生[a, b]之间的整数

randrange(a, b, step)

  • 产生[a:b:step]里的一个数
  • step默认等于1

choice(string/tuple/list)

  • 产生对象里的其中一个元素

sample(string/tuple/list/set)

  • 从对象中任选两个元素，以列表形式返回

shuffle(list)

  • 随机打乱顺序
  • 返回None
